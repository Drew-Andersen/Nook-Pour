## Getting Started

### Local Installation / Testing

1. Clone the rep

```
git clone git@github.com:Drew-Andersen/Nook-Pour.git
```

2. Install dependencies

```
npm install
```

3. Seed the database

```
npm run seed
```

4. Launch the app in development environments

```
npm run develop
```
